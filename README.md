# 🟩 BloxdTool

**Bloxd.io Utility Toolkit**

BloxdTool は、**Bloxd.io プレイヤー向けの便利ツール集**です。

現在開発中 🚧

---

# ✨ 主な機能（予定）

## 📍 名札座標推定

測定した座標と距離から、トライブメンバーの名札の位置を推定します。

### 特徴

- 複数回測定対応
- 誤差最小探索
- 高速探索アルゴリズム
- 信頼度表示
- ヒートマップ表示

---

## 🧱 クラフト素材計算機

必要個数を入力すると、必要素材を自動計算します。

例

Diamond Pickaxe ×15

↓

Diamond ×45

Stick ×30

---

## 💾 データ保存

- JSON保存
- JSON読み込み
- ローカル保存

---

## 📱 PWA対応

ホーム画面へ追加可能

オフライン対応予定
